Pendahuluan

Sistem Operasi (Operating System / OS) merupakan komponen fundamental dalam perangkat digital. OS berfungsi sebagai penghubung antara perangkat keras (hardware) dan perangkat lunak (software), memastikan semua sumber daya komputer digunakan secara efisien dan terkoordinasi.

Definisi dan Fungsi Sistem Operasi

  • Manajemen Proses: Mengatur proses yang berjalan pada CPU agar tidak terjadi konflik.
  • Manajemen Memori: Mengalokasikan ruang memori untuk aplikasi agar berjalan lancar.
  • Manajemen Storage / File System: Menyimpan dan mengelola struktur direktori.
  • Manajemen I/O: Mengatur komunikasi antara perangkat keras dan aplikasi.
  • User Interface (UI): Menyediakan antarmuka GUI/CLI agar interaksi lebih mudah.

Contoh dan Kategori Sistem Operasi

Kategori Desktop

  • Windows: OS user-friendly populer di perkantoran dan rumah.
  • Linux: OS open-source stabil dan aman, sering dipakai di server dan developer.
  • macOS: OS eksklusif Apple, populer di dunia kreatif.

Kategori Mobile

  • Android: OS fleksibel dengan banyak variasi perangkat.
  • iOS: OS eksklusif Apple yang aman dan stabil.
