Minggu 2: Sejarah dan Evolusi Sistem Operasi

Mata Kuliah Sistem Operasi 1

🎯 Tujuan Pembelajaran

  • Memahami perkembangan sistem operasi dari masa ke masa.
  • Mengenali generasi OS dan teknologi pendukungnya.
  • Menjelaskan perbedaan OS lama dan OS modern.
  • Mengaitkan evolusi OS dengan kebutuhan teknologi saat ini.

🕰️ Awal Mula Sistem Operasi

Pada tahun 1950-an, komputer belum memiliki sistem operasi seperti sekarang. Semua perintah diketik langsung ke mesin. Karena proses ini lambat dan rumit, mulai dikembangkan program yang mempermudah interaksi antara manusia dan komputer. Dari sinilah cikal bakal sistem operasi lahir.

💡 Analogi: Jika komputer adalah gedung kantor besar, sistem operasi adalah manajer yang mengatur siapa kerja di mana, kapan, dan pakai alat apa.

⚙️ Generasi Sistem Operasi

Generasi Tahun Ciri Utama Contoh
🧮 Generasi I1940–1950Tidak ada OS, kontrol manualENIAC
💾 Generasi II1950–1960Batch System (otomatis, satu per satu)IBM 704
🖥️ Generasi III1960–1970Multiprogramming, Time-sharingUNIX Awal
💡 Generasi IV1980–2000GUI, user-friendlyWindows, macOS
☁️ Generasi Modern2000–SekarangMobile & Cloud OSAndroid, iOS

🚀 Perkembangan Teknologi OS

  1. Batch → Time-Sharing → Real-Time: dari tunggu satu program ke multitasking cepat.
  2. PC → GUI → Mobile → Cloud: komputer besar menjadi OS di saku kita.
  3. Open Source Era: munculnya Linux yang gratis, aman, dan fleksibel.

🌍 OS Populer Sepanjang Waktu

  • UNIX – Pondasi OS modern (1969)
  • Windows – Populer di PC (1985)
  • Linux – Open source & stabil (1991)
  • macOS – Stabil & aman (2001)
  • Android – Dominan di mobile (2008)
  • iOS – Eksklusif Apple (2007)

🧩 Kesimpulan

  • OS berevolusi dari sistem manual ke sistem pintar dan mobile.
  • Setiap generasi hadir karena kebutuhan teknologi baru.
  • OS kini hadir di berbagai perangkat, dari PC hingga cloud.
  • Memahami evolusi OS membantu kita memahami desain OS modern.
